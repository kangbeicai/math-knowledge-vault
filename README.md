# Mathematics Knowledge Vault

一个以 **Obsidian + GitHub** 维护的个人数学知识库。

当前内容以“流形与微分几何”为第一条学习主线，但仓库结构面向全部数学主题，可逐步扩展到：

- 数学基础与逻辑
- 线性代数
- 微积分与分析
- 抽象代数
- 几何与拓扑
- 概率与统计
- 常微分方程与偏微分方程
- 数值数学与优化
- 数学物理
- 数学在工程问题中的应用

## 使用方法

1. 克隆或下载仓库。
2. 在 Obsidian 中选择 **Open folder as vault**。
3. 打开 [`Home.md`](Home.md)。
4. 从 `10 Maps/数学知识地图.md` 或某条学习路径进入。
5. 新概念先放入 `00 Inbox/`，整理后移动到正式目录。

## 设计原则

- **MOC 而非纯文件夹导航**：文件夹负责粗分类，MOC 负责表达知识关系。
- **一概念一笔记**：便于双向链接、复用和持续修正。
- **问题驱动**：重点记录“为什么如此定义”，而不只是抄写结论。
- **对象与表示分离**：区分数学对象、坐标、矩阵和数值离散。
- **允许未完成**：用状态属性明确“初见、可复述、可计算、可证明、可应用”。
- **来源可追踪**：外部文章作为地图或参考，不作为唯一权威。
