# Mathematics Knowledge Vault

一个使用 **Obsidian + GitHub** 长期维护的个人数学知识系统。

当前以“流形与微分几何”为第一条学习主线，架构面向全部数学内容，可继续扩展到分析、代数、概率、微分方程、优化、数值数学和数学物理。

## V2 特点

- **MOC 知识地图**：表达概念关系，而不只是文件目录。
- **一概念一笔记**：持续补充动机、定义、例子、反例和应用。
- **统一 Properties**：描述类型、领域、状态、前置知识和来源。
- **Obsidian Bases**：动态查看概念、问题、来源和练习。
- **问题驱动**：保留真正的理解障碍和关闭标准。
- **核心无第三方依赖**：使用 Obsidian 原生 Properties、Bases、Templates 和 MOC。

## 快速开始

1. 克隆或下载仓库。
2. 在 Obsidian 中选择 **Open folder as vault**。
3. 打开 `Home.md`。
4. 从“数学知识地图”或某条学习路径进入。
5. 临时内容先放入 `00 Inbox/`。
