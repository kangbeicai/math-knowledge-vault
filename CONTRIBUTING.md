# 知识库维护规范

## 文件命名

- 概念笔记：直接使用中文概念名。
- MOC：以 `MOC` 结尾。
- 问题笔记：尽量使用完整问句。
- 来源笔记：标题 + 标识符。
- 周复盘：`YYYY-Www 周复盘`。

## 链接规则

- 每篇正式概念笔记至少链接到一个 MOC。
- 每篇概念笔记列出必要前置知识。
- 不为“看起来相关”而滥加链接。
- 链接应表达明确关系：前置、推广、对偶、应用或易混淆。

## 内容规则

- 定义与解释分开。
- 原文摘录标注来源。
- 自己的理解允许修改。
- 未核实内容明确标记。
- 尽量给出例子和反例。

## Git 提交建议

```text
notes: add tangent space explanation
questions: refine cotangent-vector question
roadmap: extend numerical mathematics MOC
review: add 2026-W25 weekly review
```
